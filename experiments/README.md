## How Experiments Work

Each experiment follows the same structure:

1. **Hypothesis** — What we believe to be true
2. **Setup** — How we test it
3. **Execution** — Raw logs, screenshots, data
4. **Analysis** — What we learned
5. **Conclusion** — Does the hypothesis hold?

**Nothing goes into a chapter without being validated here first.**
