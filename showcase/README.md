## How to Submit

Built something using what you learned? We'd love to feature it!

1. Fork this repo
2. Create a file: `showcase/NNN-your-product-name.md` (use next available number)
3. Use this template:

```markdown
## Product Name

> One-line description

**Built by:** [Your Name] ([GitHub](link) | [Twitter](link))

**What I said to AI:**
> "Your exact prompt/command"

**What AI built:**
A [brief description of the product, features, tech stack]

**Time to build:** X hours/days
**Revenue:** $X/month (or "not monetized yet")
**Live URL:** [link](https://...)

**What I learned:**
[Key takeaway from the experience]
```

4. Open a PR using [our template](../.github/PULL_REQUEST_TEMPLATE.md)

## Rules

- ✅ Must be a real, working product (not just a tutorial exercise)
- ✅ Must be built primarily using AI coding agents
- ✅ OpenClaw + Claude Code preferred, but any AI tool combo is welcome
- ✅ Be honest about time, costs, and results
- ✅ Include a live URL if possible
